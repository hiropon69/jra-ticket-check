# JRA 万馬券チェック 公開手順

このフォルダの中身は静的サイトです。サーバー処理は不要です。

## 一番簡単: Netlify Drop

1. https://app.netlify.com/drop を開く
2. この `outputs` フォルダ、または `jra-ticket-check-public.zip` をドラッグ&ドロップ
3. 発行されたURLをiPhoneで開く

## GitHub Pages

1. GitHubで新しいリポジトリを作る
2. このフォルダ内のファイルをアップロードする
3. Settings > Pages で `Deploy from a branch` を選ぶ
4. `main` / root を選んで保存する
5. 表示されたURLをiPhoneで開く

## 注意

- 宝塚記念の出走馬は `宝塚出走馬` ボタンで読み込めます。
- 単勝想定オッズは公開ページに含めていないため、必要に応じて入力してください。
- 公開URLを知っている人はアプリを開けます。個人情報や購入履歴は入れない運用がおすすめです。
